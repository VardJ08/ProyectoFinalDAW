# Proyecto Final - Gestión de Productos

Aplicación web desarrollada en PHP y MySQL utilizando el patrón MVC.
Permite realizar operaciones CRUD sobre un catálogo de productos.

## Tecnologías
- PHP
- MySQL (XAMPP)
- HTML, CSS, JavaScript
- MVC
- InfinityFree

## Funcionalidades
- Crear productos
- Listar productos
- Ver Detalles de productos
- Editar productos
- Eliminar productos con confirmación

## Base de datos
Importar el archivo: database/database.sql

## Ejecución local
1. Copiar el proyecto en: C:\xampp\htdocs\
2. Iniciar Apache y MySQL en XAMPP
3. Acceder desde el navegador
